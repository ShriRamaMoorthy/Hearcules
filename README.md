# HEARcules

> A lightweight, single-file web app that reads URLs, PDFs and text aloud , built for learning on the go.


---

## What It Does

Paste a URL (GeeksForGeeks, Wikipedia, any article), upload a file, or paste raw text - Hearcules extracts the content and reads it to you clearly, sentence by sentence, with live highlighting. Perfect for learning while walking, commuting, or doing anything hands-free.

*Built for focused, distraction-free learning. Turn your walks into lectures.*
